# Como Pegar a Cor Exata do Site

## Método 1: DevTools (Mais Preciso)

1. Abra o site: https://levitamoveis.com.br
2. Pressione **F12** (ou **Cmd+Option+I** no Mac)
3. Clique no ícone de **seleção** (canto superior esquerdo do DevTools) ou pressione **Ctrl+Shift+C**
4. **Clique no header azul claro** (a parte superior com telefone, email, etc)
5. No painel direito, procure por:
   - `background-color`
   - `background`
   - Ou na aba **Computed** procure por `background-color`
6. Você verá algo como: `background-color: rgb(179, 217, 242)` ou `#b3d9f2`
7. **Copie o código hexadecimal** (ex: `#B3D9F2`)

## Método 2: Conta-gotas do DevTools

1. Abra o DevTools (F12)
2. Clique no elemento azul claro
3. No painel de estilos, clique no **quadrado colorido** ao lado de `background-color`
4. Clique no **conta-gotas** (ícone de pipeta)
5. Clique no azul claro do site
6. Copie o código hexadecimal que aparecer

## Método 3: Extensão do Chrome

Instale a extensão **ColorZilla** ou **Eye Dropper**:
1. Vá na Chrome Web Store
2. Instale "ColorZilla" ou "Eye Dropper"
3. Clique no ícone da extensão
4. Clique no azul claro do site
5. Copie o código hexadecimal
